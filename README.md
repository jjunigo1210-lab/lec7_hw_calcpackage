# 계산기 패키지

## 1. 패키지 구조

```text
mycalc_project/
├─ mycalc/              # 계산기 패키지
│  ├─ __init__.py       # 공개 함수 정리
│  ├─ basic.py          # 사칙연산, 나머지, 정수 나눗셈 등 기본 연산
│  ├─ scientific.py     # 제곱, 제곱근, 팩토리얼, 조합/순열 등 과학 계산
│  └─ utils.py          # 입력 검증 등 공통 유틸함수
└─ main.py              # 패키지 기능 테스트용 실행 스크립트
